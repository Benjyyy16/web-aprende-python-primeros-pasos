Aprende Python desde Cero

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Es conocido por su sintaxis simple y legible, lo que lo hace perfecto para principiantes.

Tu Primer Programa

# Este es tu primer programa en Python
print("¡Hola, Mundo!")

Este simple programa muestra el clásico "¡Hola, Mundo!" en la pantalla.

Variables y Tipos de Datos

# Variables y tipos básicos
nombre = "Ana"          # String (texto)
edad = 25              # Integer (número entero)
altura = 1.65          # Float (número decimal)
es_estudiante = True   # Boolean (verdadero/falso)

print(f"Me llamo {nombre} y tengo {edad} años")

Estructuras de Control

# Condicional if-else
edad = 18
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

# Bucle for
for i in range(5):
    print(f"Número: {i}")
